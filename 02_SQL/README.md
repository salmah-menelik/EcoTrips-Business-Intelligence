# SQL Queries

This folder contains the SQL scripts developed for the **EcoTrips Business Intelligence Project**.

The queries were written in **Google BigQuery (Standard SQL)** and support the business analyses, KPI calculations, and dashboard development presented in this repository.

## Folder Structure

### 01_data_exploration.sql
Exploratory SQL queries used to understand reservation patterns, offer categories, customer behavior, and sustainable practices.

### 02_business_queries.sql
Business-focused analyses that answer key operational and strategic questions, including revenue trends, customer repurchase behavior, and offer performance.

### 03_kpi_calculations.sql
SQL scripts used to calculate the key performance indicators (KPIs) displayed in the Power BI dashboards.

## Notes

The original EcoTrips dataset was provided in **Portuguese**, so the database schema (table names and column names) remains in Portuguese to preserve compatibility with the source data.

However, all documentation, comments, query titles, dashboard labels, and repository content have been translated into **English** to make the project accessible to an international audience.

**Environment**
- Google BigQuery
- Standard SQL
