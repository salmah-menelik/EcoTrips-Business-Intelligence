# 📚 Documentation

This folder contains the supporting documentation for the **EcoTrips Business Intelligence Project**. The documents provide additional context about the project's objectives, methodology, business requirements, and data structure.

## Contents

- **Executive_Summary.pdf** – Executive overview of the project, including business objectives, analytical approach, key findings, and recommendations.
- **Project_Overview.md** – Overview of the project's objectives, workflow, and technologies used throughout the development process.
- **Business_Questions.md** – Key business questions addressed through SQL analysis and Power BI dashboards.
- **Data_Dictionary.md** – Description of the datasets, tables, and entities used in the project.

## Purpose

The documentation complements the technical deliverables by explaining the business context, analytical methodology, and project design. It is intended to help both technical and non-technical
