# Project Overview

## Objective

EcoTrips is a Business Intelligence project developed to analyze customer behavior, reservation trends, revenue performance, and sustainability initiatives for a travel company.

The project combines SQL, Power BI, and data modeling techniques to transform raw operational data into actionable business insights.

## Project Workflow

1. Data Modeling
2. SQL Analysis
3. KPI Development
4. Dashboard Design
5. Business Recommendations

## Technologies

- Google BigQuery
- SQL (Standard SQL)
- Power BI
- DAX
- Power Query
- dbdiagram.io
