# Dashboard Images

This folder contains screenshots of the Power BI dashboards developed for the **EcoTrips Business Intelligence Project**.

The images provide a quick overview of the interactive reports without requiring Power BI Desktop.

## Dashboard Highlights

- Executive business overview
- Revenue and reservation analysis
- Customer behavior insights
- Sustainability performance
- Key Performance Indicators (KPIs)

The complete interactive dashboard is available in the `03_PowerBI` folder.
