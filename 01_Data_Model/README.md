# Data Model

This folder contains the database schema developed for the **EcoTrips Business Intelligence Project**.

The data model was designed using a relational structure to support SQL analysis and Power BI reporting.

## Files

- **EcoTrips.dbml** – Source file created with DBML (Database Markup Language).
- **EcoTrips_Data_Model.pdf** – PDF version of the entity relationship diagram.
- **EcoTrips_Data_Model.png** – Image version of the entity relationship diagram.

The data model illustrates the relationships between the main entities used throughout the project, including customers, reservations, travel offers, and sustainability practices.
