# Project Presentation

This folder contains the presentation developed for the **EcoTrips Business Intelligence Project**.

The presentation summarizes the project's objectives, methodology, business questions, analytical approach, and key insights obtained through SQL analysis and Power BI dashboards.

## Contents

- **EcoTrips_Presentation.pdf** – Final presentation in PDF format.
- **EcoTrips_Presentation.pptx** – Editable presentation file.

## Presentation Highlights

- Business problem and project objectives
- Data model overview
- SQL analysis
- KPI development
- Dashboard walkthrough
- Business insights and recommendations

The presentation was prepared in English to make the project accessible to an international audience.
