# Power BI Dashboard

This folder contains the Power BI dashboard developed for the EcoTrips Business Intelligence Project.

## Dashboard Overview

The report was designed to support business decision-making by providing insights into:

- Revenue performance
- Customer behavior
- Reservation trends
- Offer performance
- Sustainability metrics
- Key business KPIs

## Tools Used

- Microsoft Power BI Desktop
- Power Query
- DAX
- Star Schema Data Model

## File

- **EcoTrips.pbix** – Complete Power BI report containing all dashboards, measures, relationships, and visualizations.
